This guide will help you deploy a Bun application on Zerops quickly.

## Prerequisites

- A [Zerops account](https://app.zerops.io)
- Basic knowledge of Bun

## Deploy Using a Recipe

<Steps>
  <Step title="Log in to Zerops">
    Navigate to [app.zerops.io](https://app.zerops.io).
  </Step>
  
  <Step title="Import the project">
    Click **Import a project** and paste:
    
    ```yaml
    project:
      name: my-bun-app
      tags:
        - bun
    
    services:
      - hostname: api
        type: bun@1.1
        enableSubdomainAccess: true
        buildFromGit: https://github.com/zeropsio/recipe-bun
    
      - hostname: db
        type: postgresql@16
        mode: NON_HA
        priority: 1
    ```
  </Step>
  
  <Step title="Wait for deployment">
    Zerops will build and deploy automatically (2-3 minutes).
  </Step>
  
  <Step title="Access your app">
    Open the subdomain URL from the **IP addresses & Public Routing** section.
  </Step>
</Steps>

## Deploy Your Own Bun App

### 1. Create a Bun Service

<Tabs>
  <Tab title="Using GUI">
    1. Click **Add new service**
    2. Select **Bun**
    3. Choose version (1.0, 1.1, or latest)
    4. Set hostname (e.g., "api")
  </Tab>
  
  <Tab title="Using zCLI">
    ```yaml title="description.yaml"
    project:
      name: my-project
    
    services:
      - hostname: api
        type: bun@1.1
        minContainers: 1
        maxContainers: 6
    ```
    
    ```bash
    zcli project project-import description.yaml
    ```
  </Tab>
</Tabs>

### 2. Create Your Bun Application

Initialize a new Bun project:

```bash
bun init
```

Create a simple server:

```typescript title="server.ts"
import { serve } from "bun";

const server = serve({
  port: 3000,
  fetch(req) {
    return new Response("Hello from Bun on Zerops!");
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

### 3. Add zerops.yaml

```yaml title="zerops.yaml"
zerops:
  - setup: api
    build:
      base: bun@1.1
      buildCommands:
        - bun install
        - bun build ./server.ts --outdir ./dist
      deployFiles:
        - dist
        - node_modules
        - package.json
      cache: node_modules
    
    run:
      base: bun@1.1
      ports:
        - port: 3000
          httpSupport: true
      start: bun run dist/server.js
```

### 4. Deploy

<Tabs>
  <Tab title="From Git">

    2. In Zerops: **Deploy** â†’ **From Git**
    3. Enter repository URL
  </Tab>
  
  <Tab title="Using zCLI">

    ```bash
    zcli push
    ```
  </Tab>
</Tabs>

## Connect to PostgreSQL

### Add Database Service

```yaml
services:
  - hostname: db
    type: postgresql@16
    mode: NON_HA
```

### Install PostgreSQL Client

```bash
bun add pg
```

### Use in Your App

```typescript title="server.ts"
import { serve } from "bun";
import { Pool } from "pg";

const pool = new Pool({
  host: "db",  // Service hostname
  port: 5432,
  user: "db",
  password: process.env.DB_PASSWORD,
  database: "db",
});

const server = serve({
  port: 3000,
  async fetch(req) {
    const result = await pool.query("SELECT NOW() as time");
    
    return new Response(
      JSON.stringify({ time: result.rows[0].time })
    );
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

<Info>
  Services in the same project communicate via private network using hostnames.
</Info>

<Tabs>

## Build Commands

Common Bun build patterns:

<Tabs>
  <Tab title="Simple build">
    ```yaml
    buildCommands:
      - bun install
      - bun build ./server.ts --outdir ./dist
    ```
  </Tab>
  
  <Tab title="With TypeScript">
    ```yaml
    buildCommands:
      - bun install
      - bun build ./src/index.ts --outdir ./dist --target bun
    ```
  </Tab>
  
  <Tab title="With tests">
    ```yaml
    buildCommands:
      - bun install
      - bun test
      - bun build ./server.ts --outdir ./dist
    ```
  </Tab>
</Tabs>

## Environment Variables

<Tabs>
  <Tab title="In GUI">
    1. Go to service settings
    2. **Environment Variables**
    3. Add variables
  </Tab>
  
  <Tab title="In zerops.yaml">
    ```yaml
    run:
      envVariables:
        NODE_ENV: production
        PORT: 3000
        API_URL: https://api.example.com
    ```
  </Tab>
</Tabs>

Access in your app:

```typescript
const apiUrl = process.env.API_URL;
const port = process.env.PORT || 3000;
```

## Running Scripts

Define scripts in `package.json`:

```json title="package.json"
{
  "name": "my-bun-app",
  "scripts": {
    "start": "bun run server.ts",
    "dev": "bun --watch server.ts",
    "build": "bun build ./server.ts --outdir ./dist",
    "test": "bun test"
  }
}
```

Use in zerops.yaml:

```yaml
run:
  start: bun start
```
